# Security Policy

This repository goes through static code analysis and manual review for security issues/vulnerabilities.

## Reporting Process

Please [report an issue][create-issue] if there is an issue or security vulnerability.
The maintainers of this project will review.

Please include the following information:
- A description of the vulnerability and potential impact.
- A PoC (Proof of Concept) or steps to reproduce.
- The affected software versions.

## Supported Versions

Only the latest release of the `main` branch is supported.

[create-issue]: https://github.com/logicmonitor/logicmonitor.edwin_ai/issues/new
